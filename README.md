Futuron: Lightcycle AI Battles
==============================

Getting started
---------------

Prerequisites:
* [Git](http://git-scm.com/)
* [Node.js](http://nodejs.org/)
* [NPM](https://npmjs.org/)
* [Grunt](http://gruntjs.com/)
* [Bower](http://bower.io/)

First steps:
* Clone the repository
* Run `npm install`
* Run `bower install`
* Run `grunt server --force` (--force is unnecessary if you have Compass)
